# sans blog

1. 本地需要先启动mongodb 记得要关闭 brew services start mongodb-community
2. 项目中使用了next `new NextApp({ dir: 'app' })`
3. 有个bug nextapp 必须在 static app之后 不然staticapp 会不生效

## 安装

```
  npm i  
```
